# Deployment Helper

[![codecov](https://codecov.io/gh/onlishop/deployment-helper/graph/badge.svg?token=9F9GYJ3OWS)](https://codecov.io/gh/onlishop/deployment-helper)
[![PHP](https://github.com/onlishop/deployment-helper/actions/workflows/php.yml/badge.svg)](https://github.com/onlishop/deployment-helper/actions/workflows/php.yml)

This is a helper script to install or update Onlishop on the target system. 
It's independent of the Onlishop version and can be used for all versions newer 6.5.

## Installation

```bash
composer require onlishop/deployment-helper
```

[For docs see here](https://developer.onlishop.com/docs/guides/hosting/installation-updates/deployments/deployment-helper.html#configuration)
